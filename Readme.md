## Leitura de Arquivos e Conversão para XML
## Sobre o projeto

Este projeto foi desenvolvido como atividade prática do curso de Java da EBAC (Escola Britânica de Artes Criativas e Tecnologia).

O objetivo é realizar a leitura de um arquivo texto contendo dados estruturados, converter cada linha em um objeto Java e exibir essas informações em formato XML.

## Tecnologias utilizadas

Java
API de Entrada e Saída de Arquivos (java.io)
Programação Orientada a Objetos

## Funcionalidades
Leitura de um arquivo texto (.txt).
Separação dos dados utilizando um delimitador (;).
Conversão de cada linha em um objeto Java.
Exibição dos objetos em formato XML no console.

## Objetivo

Praticar a leitura de arquivos em Java, a manipulação de dados, a criação de objetos a partir de informações lidas de um arquivo e a representação desses objetos em formato XML, conforme proposto na atividade da EBAC.
